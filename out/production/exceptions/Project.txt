ETAPA 1: fazer um programa para ler os dados de uma reserva de hotel (número do quarto, data
de entrada e data de saída) e mostrar os dados da reserva, inclusive a sua duração em
dias.

ETAPA 2: em seguida, ler novas datas de entrada e saída, atualizar a reserva, e mostrar
novamente a reserva com os dados atualizados.

ETAPA 3: O programa não deve aceitar dados inválidos para a reserva, conforme as seguintes regras:
— Alterações de reserva só podem ocorrer para datas futuras
— A data de saída deve ser maior que a data de entrada
